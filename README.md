# Portfolio

A Next.js portfolio website showcasing my professional experience, projects, and baking recipes.

## Features

- **Experience**: Work history and professional background
- **Projects**: Technical projects and developments
- **Recipes**: Collection of baking recipes

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Getting Started
```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build
```bash
pnpm build
pnpm start
```
